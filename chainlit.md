# Trợ lý Knowledge nội bộ (RAG)

Hỏi đáp trên tài liệu nội bộ của tổ chức.

- **Hỏi**: gõ câu hỏi, agent tìm trong kho tri thức và trả lời kèm nguồn.
- **Nạp tài liệu**: đính kèm file (pdf, docx, xlsx, pptx, ...) — tự động chuyển sang markdown và index.
- **Personalize**: đưa feedback về cách trả lời ("luôn trả lời ngắn gọn") — agent ghi nhớ qua các phiên.
